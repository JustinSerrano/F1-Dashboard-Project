COMP 3512 - F1 Dashboard Project

This project is a Single-Page Application (SPA) for viewing Formula 1 (F1) race data. The application allows users to browse races, view qualifying and race results, and explore details about circuits, drivers, and constructors.

Features

  • Home View: Brief description of the application and technologies used.
  • Races View: Displays races sorted by round for a selected season, with sorting capabilities.
  • Dialogs/Popups:
    • Driver Details: Information about drivers and their race results.
    • Constructor Details: Information about constructors and their race results.
    • Circuit Details: Information about circuits.
  • Favorite functionality for circuits, drivers, and constructors.

Technologies Used

  • HTML5: Markup for all views is contained in index.html.
  • CSS3: Custom styling for layout, modals, and animations.
  • JavaScript (ES6): DOM manipulation, event handling, API interaction.
  • GitHub Pages: Hosting the application.
